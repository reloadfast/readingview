## Search Tips

**Do:**
- ‚úÖ Use simple, clear terms
- ‚úÖ Try author name only first
- ‚úÖ Search series name for all books

**Don't:**

## Understanding Open Library Data

### What Open Library Knows

‚úÖ **Available:**
- Book titles and authors
- Publication history
- Multiple editions
- Cover images
- Series information (sometimes)
- ISBNs and identifiers

‚ùå **Not Available:**
- Official future release dates
- Pre-order information
- Publisher announcements
- Upcoming books not yet published

### Where to Get Release Dates

Since Open Library doesn't have future dates, get them from:

1. **Publisher websites**
   - Tor.com, Penguin Random House, etc.
   - Official announcements

2. **Author websites/social media**
   - Twitter announcements
   - Blog posts
   - Newsletters

3. **Goodreads**
   - Community-maintained dates
   - Use as reference (link it!)

4. **Amazon**
   - Pre-order pages
   - Release date estimates

5. **Fan wikis**
   - Series-specific wikis
   - Community tracking

## Troubleshooting

### "Search returned no results"

**Solutions:**
1. Try broader terms
2. Search author only
3. Check spelling
4. Use "General" search type
5. Try manual entry with helper link

### "Wrong book in results"

**Identify correct book by:**
- Publication year
- Author name spelling
- Cover image
- Click "üîó Open Library" to verify

### "Can't find upcoming book"

**Remember:**
- Open Library shows *published* books
- Upcoming books may not be there yet
- Use manual entry for brand new announcements
- Add Open Library link later when available

### "Author match not working"

**Manual steps:**
1. Select "<Create New Author>"
2. Type exact name
3. Continue with form
4. System will track both spellings
